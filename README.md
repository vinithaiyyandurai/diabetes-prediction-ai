# Diabetes Risk Prediction with Explainable AI

## Overview
Machine learning system to predict diabetes risk
using patient health data with SHAP explainability.

## Results
- Accuracy : 91%
- ROC-AUC  : 0.93
- Model    : Gradient Boosting
- Dataset  : 768 patients (Pima Indians)

## Tech Stack
Python | scikit-learn | SHAP | Streamlit | pandas | numpy

## Features
- Predicts diabetes risk (0-100%)
- SHAP explanation for every prediction
- Interactive sliders for patient data
- Live web app

## Project Steps
1. Data Collection & EDA
2. Data Cleaning & Preprocessing
3. Model Training (3 models compared)
4. Hyperparameter Fine-tuning
5. SHAP Explainable AI
6. Streamlit Deployment
